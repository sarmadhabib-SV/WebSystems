About Amoura Vision

We help teams detect surface defects, misalignment, and labeling errors using a fast CV pipeline and a clean, explainable UI.

This right-floating block satisfies the project float requirement.

What we do

Amoura Vision is designed for “last-mile” quality checks. Instead of making operators guess, we surface a clear decision signal (confidence, category, and a short explanation).

For this class project, the site is fully static and demonstrates front-end behaviors only. In a real product, images would be sent to an inference service, and results would be stored for reporting.

Defects, labels, alignment
Deployment
Edge + Cloud

How the pipeline works (concept)

  1. Capture an image from a fixed camera over the line.
  2. Normalize lighting / crop region-of-interest.
  3. Run a detection/classification model.
  4. Return the top signals with confidence.
  5. Store results for dashboards and continuous improvement.
